RecFix - Installation Guide
==========================

Thank you for downloading RecFix! This guide will help you install the extension in your preferred browser.

For Chrome Users:
----------------
1. Unzip the downloaded file if you haven't already
2. Open Chrome and go to chrome://extensions/
3. Enable "Developer mode" (toggle switch in the top-right corner)
4. Click "Load unpacked"
5. Navigate to the unzipped RecFix folder and select it
6. Click the puzzle piece icon in Chrome's toolbar
7. Pin RecFix to your toolbar for easy access

For Firefox Users:
----------------
1. Unzip the downloaded file if you haven't already
2. Open Firefox and go to about:debugging
3. Click "This Firefox" in the left sidebar
4. Click "Load Temporary Add-on"
5. Navigate to the unzipped "Firefox Extension" folder
6. Select the manifest.json file
7. Click the extensions icon in Firefox's toolbar to pin RecFix

Troubleshooting:
---------------
Chrome:
- If the extension doesn't appear, try refreshing the extensions page
- Make sure all files are properly unzipped
- Try disabling and re-enabling Developer mode
- Restart Chrome if needed

Firefox:
- Extension will need to be reloaded after browser restart
- Make sure you're selecting the correct manifest.json from the Firefox Extension folder
- Check the Browser Console (Ctrl+Shift+J or Cmd+Shift+J) for any error messages
