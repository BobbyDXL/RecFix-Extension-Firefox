RecFix for Firefox - Installation Guide
==================================

Thank you for downloading RecFix! This guide will help you install the extension in Firefox.

Installation Steps:
-----------------
1. Unzip the downloaded file if you haven't already
2. Open Firefox and go to about:debugging
3. Click "This Firefox" in the left sidebar
4. Click "Load Temporary Add-on"
5. Navigate to the unzipped RecFix folder
6. Select the manifest.json file
7. Click the extensions icon in Firefox's toolbar to pin RecFix

Important Notes:
--------------
- This is a temporary installation that will need to be reloaded after browser restart
- The extension will remain active until Firefox is closed
- You'll need to reload the extension each time you restart Firefox
- This is standard behavior for developer versions of Firefox extensions

Troubleshooting:
---------------
- If the extension doesn't appear:
  * Make sure you're selecting the correct manifest.json file
  * Check the Browser Console (Ctrl+Shift+J or Cmd+Shift+J) for error messages
  * Try reloading the extension
  * Restart Firefox if needed
- If you encounter any issues with the extension:
  * Check that all files were properly unzipped
  * Verify you're using the Firefox-specific version of the extension
  * Look for error messages in the Browser Console
